# 2DRocketGame
날아오는 운석을 피하는고 부시는 게임

## 개발
- 엔진 : Unity
- 언어 : C#

## 조작방법
- 이동 : 터치
- 공격 : 터치

## 게임 특징
- 터치패드를 이용한 이동 구현으로 모바일에서 이용가능
